Как создать урок за 2 клика?

    {% if course_id %}
  1. Перейдите по ссылке и нажмите кнопку + Добавить урок сверху справа
  2. Выберите режим Сгенерировать урок и введите тему.
    3. {% else %}
  3. Перейдите на главную страницу
  4. Нажмите кнопку + Класс
  5. Введите название курса
  6. На странице уроков нажмите кнопку + Добавить урок сверху справа
  7. Выберите режим Сгенерировать урок и введите тему.
    8. {% endif %}

Как сгенерировать раздел по своему плану?

    {% if lesson_id %}
  1. Перейдите по ссылке и нажмите кнопку + Добавить раздел (на компьютерах - справа под списком разделов, на телефонах - снизу страницы)
  2. Выберите тип раздела
  3. Адаптируйте шаблон так, как вам хочется его видеть
    4. {% else %}
  4. Создайте любой урок, откройте его и нажмите кнопку + Добавить раздел (на компьютерах - справа под списком разделов, на телефонах - снизу страницы)
  5. Выберите тип раздела
  6. Адаптируйте шаблон так, как вам хочется его видеть
    7. {% endif %}

Важно:

При генерации раздела аудио не озвучивается автоматически.

Вы должны нажать кнопку Редактировать задание , затем выбрать Озвучить .

Как создавать задания по материалам учебника?

    {% if lesson_id %}
  1. Перейдите по ссылке и нажмите кнопку + Добавить задание внизу страницы
  2. Выберите тип задания - PDF
  3. Загрузите PDF файл учебника
  4. Сохраните
  5. Выберите до 3 страниц, задания с которых вы хотите взять. Для этого сверху справа страницы нажмите кнопку
  6. Нажмите на кнопку и кнопку Сгенерировать
    7. {% else %}
  7. Создайте любой урок и откройте его
  8. Выберите тип раздела
  9. Адаптируйте шаблон так, как вам хочется его видеть
    10. {% endif %}

Важно:

Генерация заданий по учебнику доступно только в режиме создания курса (недоступно в режиме виртуального класса)
